---
title: "Wallet transaction object"
description: "This object represents a wallet transaction object."
---

## Schema[](#schema "Direct link to heading")

```json
{
  "wallet_transaction": {
    "lago_id": "b7ab2926-1de8-4428-9bcd-779314ac1111",
    "lago_wallet_id": "12345",
    "status": "settled",
    "transaction_type": "inbound",
    "amount": "10.0",
    "credit_amount": "10.0",
    "settled_at": "2022-04-29T08:59:51Z",
    "created_at": "2022-04-29T08:59:51Z"
  }
}
```

| Attributes                                                                    | Description                                                                                          |
| ----------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| **lago_id** `String` Not null                                                 | Unique identifier of the wallet transaction in Lago application.                                     |
| **lago_wallet_id** `String` Not null                                          | Internal Lago wallet ID.                                                                             |
| **status** `String` Not null                                                  | Status. **Possible values**\- pending\- settled                                                      |
| **transaction_type** `String` Not null                                        | Transaction type. **Possible values**\- inbound (increasing balance)\- outbound (decreasing balance) |
| **credit_amount** `String` Not null                                           | Credits amount.                                                                                      |
| **amount** `String` Not null                                                  | Amount based on rate and currency.                                                                   |
| **created_at** `String` Not null <br /> <sub>_ISO 8601 datetime in UTC_</sub> | Date of wallet transaction creation.                                                                 |
| **settled_at** `String` <br /> <sub>_ISO 8601 datetime in UTC_</sub>          | Date the transaction was settled.                                                                    |
